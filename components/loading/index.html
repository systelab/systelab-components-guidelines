Loading

Overview

Loading spinners are used when retrieving data or performing slow computations. They notify to the user that their request is being processed. Although they do not provide details about what is occurring on the back-end, they reassure the user that their action is being processed.

Use a loading spinner whenever the wait time is anticipated to be longer than three seconds.

When to use

Loading used with an overlay. If this option is used, the whole screen will be locked and the user will not be able to interact until the loading process is finished.
We can use the load indicator on specific components so that we do not block the whole screen for the user. eg. loading cards information
We can also add it in the field validation to indicate to the user that the field is checking the information.
