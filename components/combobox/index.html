Combobox

The combobox component allows users to choose one option from a list. It is used in forms for users to submit data.

Overview

Combobox is a type of input that is used in forms, where a user is submitting data and chooses one option from a list.

Content

Labels

  • Labels inform users what to expect in the list of combobox options.
  • Keep the label short and concise by limiting it to a single line of text.

Helper text

  • Helper text is pertinent information that assists the user in choosing the right selection from the combobox menu.
  • Helper text is always available when the combobox is focused and appears underneath the label.
  • Use sentence-style capitalization and write the text as full sentences with punctuation, unless space is limited

Placeholder text

  • Placeholder text is presented in the field by default if no selection has been made from the combobox.
  • Use clear placeholder text for the combobox trigger so that users understand the purpose.
Use a placeholder in case the combobox does not have a label above it.

Options in a menu

  • Describe the combobox option succinctly in one line of text.
  • Avoid use decorative images or icons within a combobox
We recommend presenting the options in alphabetical order.

Overflow content

Avoid having multiple lines of text in a combobox. If the text is too long for one line, add an ellipsis (…) for overflow content, and accompany it with a browser-based tooltip to show the full string of text.

Filtering

After typing text in the field, the close (x) icon appears to the right of the text in the field. This clears any text that’s been entered in the field.

Selected option

Selecting an option closes the menu and the selected option replaces the placeholder text.

Multiple selection

Use when you can select multiple options from a list or to filter information.

  • Once options have been selected from the menu, the option value appears concatenated in combobox value.
  • The menu does not close once the user makes selections. Because multiple selections are possible, the user needs to click outside of the dropdown or on the parent element to close the menu.
