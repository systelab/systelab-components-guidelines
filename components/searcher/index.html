Searcher

Searcher offers users a way to explore and select an item from a data table also aid the user in finding content.

Overview

Searcher input

The searcher input has the same attributes as the text input. The only difference is that it has a button, right aligned, that trigger the search dialog.

When the user find and select the data.

Searcher dialog

The searcher dialog allow the user to select an item from a table, also can filter the data When the user find and select the data.

When to use

  • When the user has to select an item from a list of items with a large volume of data
  • When the data table is collected from a db request and may take some time to be rendered.
