Icons

Icons are visual symbols used to represent ideas, objects, or actions. They communicate messages at a glance, afford interactivity, and draw attention to important information.

Overview

Sizing

Werfen components typically use icons on 16px artboards. Icons on 20px, 24px, and 32px artboards can also be used within the UI. Be sure icon size is consistent throughout your product.

16px
20px
24px
32px

16px and 20px icons are optimized to feel balanced when paired with 14pt and 16pt Lato Font.

Do use the correct icon size with font.
Don't use large icons with small text. Use the correct size ratio.
Use 24px and 32px when larger icons are needed.

Colors

If using the icon only the color system guide has to be followed, if using an icon + text on a button the icon has to follow the button specifications.

Use the same color for text and icons.
Don't use different colors for text and icons.

Alignment

When used next to text, icons should be center-aligned.

Align icons and text.
Don't baseline-align icons to the text.
