List box

Listbox present a list of options from which a user can select one option, or several.

Overview

Allows the user to select one or multiple options from a list

When to use

  • Can be used in forms on full pages, in modals, or on side panels
  • Utilizar si són pocas opciones y són facilmente analizables para el usuario, en caso contrario utilizar el dropdown

When not to use

  • If there are many options and the user need to scroll or find the element could be a mess
Sempre que l'espai ho permeti es preferible utilitzar un listbox en comptes d'un dropdown. Millora l'experiència d'usuari.
