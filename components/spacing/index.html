Spacing

Spacing is an important — and often overlooked — part of product design. Provides tokens to take the guesswork out of spacing and help deliver clear, functional layouts

Overview

Spacing is the negative area between elements and components. It is commonly controlled in code with margin and padding. Systelab components offers several methods such as tokens and layout components to make implementing spacing in product easier and more consistent.

Spacing scale

Use the spacing scale when building individual components. It includes small increments needed to create appropriate spatial relationships for detail-level designs. This scale is applied and used within all Systelab components.

Token rem px Example
$spacing-01 0.125 2
$spacing-02 0.25 4
$spacing-03 0.5 8
$spacing-04 0.75 12
$spacing-05 1 16
$spacing-06 1.5 24
$spacing-07 2 32
$spacing-08 2.5 40
$spacing-09 3 48
$spacing-10 4 64
$spacing-11 5 80
$spacing-12 6 96
