Chips

Chips can show multiple interactive elements together in the same area, such as a list of selectable times. There are four types of chips: input, choice and filter.

Input chip
Choice chip
Filter chip

Overview

Chips allow users to enter information, make selections or filter content. While buttons are expected to appear consistently and with familiar calls to action, chips should appear dynamically as a group of multiple interactive elements

Types

Input choice

Input chips represent information used in fields, such as an entity or different attributes.

Behaivour:
  • You can select multiple choice chip.
  • The chip could be removed by the close icon.

Choice chip

In sets that contain at least two options, choice chips represent a single selection.

Behaivour:
  • Selecting a single choice chip automatically deselects all other chips in the set.

Filter chip

Filter chips represent filters for a collection. E.g. used to filter tables, cards or lists.

You can select multiple choice chip.
Don't use a filter chip with only a single option.
States:
  • Choice and filter chips have two states, enabled and selected
