Spy menu

Overview

A spy menu is a fixed navigation menu on a webpage that remains visible and in the same position as the user scrolls down and moves about a site.

Benefits

  • Helps the user to navigate in screens where there is a large volume of information.
  • Helps to organise information within the screen

When to use

  • When the user does not see all the information on a page and scrolling is necessary to see all the content.
  • On pages with a large number of fields

When not to use

  • When the user sees all the information on a page without scrolling
