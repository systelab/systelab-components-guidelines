Percentage indicator

Percentage indicator offer a way to visualize data in an intuitive and easy way.

Overview

Percentage indicator is not a process loading indicator, use progress bar instead.

When to use

  • To visually the state of a value e.g: the state of bulks, % of connected machines

When not to use

  • When it is a currently running process, in real time

Sizing

Size
Large

In the large size, the percentage indicator and the label are shown.

Medium

Medium size only shows the percentage value, this size allows the component to be used in small containers, such as sidebars or small cards.

Small

The small version only show the visual indicator without any text indication. It is designed to be used in small spaces such as tables.

Placement

Dashboard
Sidebar
Card
Table
