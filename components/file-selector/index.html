File selector

File selector allow users to select one or more files to upload to a specific location.

Overview

File uploaders allow users to upload content of their own. A file uploader is commonly found in forms, but can also live as a standalone element.

For the file selector layout follow the form style guide

Anatomy

File selector follow the same pattern as input text

1. Label: Informs users about the purpose of the upload section. "Upload files" is the default text that appears with the file uploader.
2. Button: Button labels should be concise and describe the action that will be taken upon click. "Add file" is the default text that appears with the file uploader.
3. Uploaded file: A file that has successfully been uploaded. It appear as chip component
4. Helper text: Use to inform to the user of the specificaitons of files that could upload, e.g size or extension

Overflow

In case the number of files does not fit in the input, the input will make a line break in order to contain all the chips.

Behaivour

  • When a file is selected it appears as a chip with the file name.
  • You can delete the file with the chip icon.
